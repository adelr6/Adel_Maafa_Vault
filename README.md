# Adel_Maafa_Vault
Valorant Knowledge Vault (Obsidian Project)

This project is an Obsidian Vault dedicated to Valorant. It serves as a structured and educational knowledge base about the game.
The vault covers maps, agents, game modes, the ranked system, VCT esports, and general gameplay concepts. The objective is to present organized, clear, and interconnected information using Obsidian features such as internal links, tables, images, and structured notes.

Project Objectives

Build a complete and well-structured knowledge base about Valorant

Demonstrate organization, digital methodology, and documentation skills

Practice Markdown structuring and knowledge linking inside Obsidian

Provide content that is practical, readable, and easy to navigate

Vault Structure

The vault is organized into several folders to ensure clarity and logical navigation:

Valorant Vault
│
├── Agents
│   ├── Duelists
│   ├── Initiators
│   ├── Sentinels
│   └── Controllers
│
├── Maps
│
├── Game Modes
│
├── Competitive & Ranked
│
├── VCT (Esports)
│
└── Images & Icons


Each note is connected through Obsidian wiki links to facilitate navigation.

Content Overview

The vault contains:

General presentation of Valorant

Detailed pages for maps with structure, areas, strategy, and playstyle information

Complete individual agent pages

Explanation of Competitive Mode

Explanation of the Ranked system and ranks

VCT explanation and major teams

Structured tables and formatted notes

Locally stored icons and images to ensure they remain visible after zipping the vault

Tools and Features Used

Obsidian Markdown

Internal linking using [[Note Links]]

Tables and structured formatting

Local image storage inside the vault

CSS customization when useful for presentation

How to Use the Vault

Download or clone the repository

Extract the vault if it is zipped

Open Obsidian

Select “Open Folder as Vault”

Choose this project folder

Browse and use the notes normally

No external dependencies are required, and all images are included locally.
