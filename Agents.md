**Agents** are the playable characters in VALORANT, representing an agent of the VALORANT Protocol. Each agent serves as a different class with four abilities.

## Usage and Availability

All agents have 100 base HP (increased with shields by up to 50) and four abilities, including a Signature and an Ultimate ability.

Five agents are immediately available for players to use; Brimstone, Jett, Phoenix, Sage and Sova 

Other agents can be unlocked through the following methods:


- One agent for 1000 Valorant points or 8000 kingdom credits or a free agent token 
- - Up to 2x Free Agent Tokens can be acquired from Basic Training Event
- During Agent Recruitment Event which lasts for 28 days from the new Agent's launch, when the player accumulate 200,000 XP during the event duration, the Agent is unlocked for free
- Use of Kingdom Credits and Free Agent Tokens are disabled when Agent Recruitment Event is active

## Roles

Depending on their abilities and playstyle, agents are assigned to one of four roles determined by the game

There are currently 6 Controller agents, 8 Duelist agents, 7 Initiator agents and 7 Sentinel agents in the roster, bringing to a total of 28 playable agents overall
